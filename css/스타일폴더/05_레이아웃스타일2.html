padding margin border 방향성

            000 - padding, margin, border 를 나타냄

            000-top    : 크기 → 위쪽 방향 크기 지정
            000-bottom : 크기 → 아래쪽 방향 크기 지정
            000-left   : 크기 → 왼쪽 방향 크기 지정
            000-right  : 크기 → 오른쪽 방향 크기 지정
            
            단, border(선) 속성값은 [크기 모양 색상] 형식
            예를 들어
            오른쪽 선은 도트모양으로 10px 빨간색
            border-right : 10px dotted red

padding, margin, border-width 크기 한번에 지정하기

            padding | margin | border-width : 속성 값 크기 설정
            속성 값 1개 : 상하좌우
            padding : 상하좌우 20px 
            margin  : 상하좌우 20px 
            
            속성 값 2개 : 상하 좌우
            padding : 상하 20px 좌우 10px
            margin  : 상하 20px 좌우 10px 

            속성 값 3개 : 상 좌우 하
            padding : 상 20px 좌우 10px 하 30px
            margin  : 상 20px 좌우 10px 하 30px

            속성 값 4개 : 상 우 하 좌 (시계방향)
            padding : 상 10px 우 20px 하 30px 좌 40px
            margin  : 상 10px 우 20px 하 30px 좌 40px

margin : 0 auto;

            다른 태그와의 간격을 자동으로 지정해서 가운데 정렬하게 만드는 속성 값
            
            기본적으로 자동 정렬 (= 가운데 정렬)
            display : block 값에서 자주 사용
            
            inline, inline-block 은 text-align : center; 이용
